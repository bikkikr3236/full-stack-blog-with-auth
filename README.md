# 📝 FullStack Blog App with React, Redux & Django

A modern Full Stack Blog Application built using **React.js**, **Redux**, **Django**, and **Django REST Framework**. This project provides a complete blogging platform with secure authentication, blog management, and a responsive user interface.

---

## 🚀 Features

- 🔐 User Authentication (Login & Register)
- 👤 User Profile Management
- 📝 Create, Edit & Delete Blog Posts
- 📖 Read Blogs
- ❤️ Like & Comment System
- 🔍 Search Blogs
- 📱 Fully Responsive UI
- ⚡ REST API Integration
- 🔒 Secure Authentication using Django

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Redux
- React Router
- Axios
- Bootstrap / CSS

### Backend
- Django
- Django REST Framework
- SQLite (Default)

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/YashMarmat/FullStack_Blog_App_with_React_Redux_and_Django.git
```

```bash
cd FullStack_Blog_App_with_React_Redux_and_Django
```

---

### 2. Backend Setup

```bash
cd backend

python -m venv venv
```

### Windows

```bash
venv\Scripts\activate
```

### Linux/Mac

```bash
source venv/bin/activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Migrations

```bash
python manage.py migrate
```

Start Django Server

```bash
python manage.py runserver
```

---

### 3. Frontend Setup

```bash
cd frontend

npm install

npm start
```
